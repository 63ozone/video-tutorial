GITHUB COMMANDS:
	
-Get the repository from the github website:
  *git clone https://github.com/63ozone/video-tutorial

-Change directory into the repository folder
 *cd video-tutorial

-Tell status of the repository. Any file names in red have not been
 officially added to the repository 
 *git status

-Add the file to the repository (if you git status now, the filename will be green)
 *git add filename.txt

-officially commit the file (make changes permanent). Forces you to make a comment
 *git commit -m "comment about what you have done to the file"

-export the file (upload it) to the online repository on gitHub.com
 *git push
